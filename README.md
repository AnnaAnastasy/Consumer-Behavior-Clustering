# Consumer Behavior Cluster Analysis with KMeans

This project analyzes customer data to identify distinct behavioral clusters using the KMeans algorithm, enabling businesses to target and retain specific customer segments effectively.

## Features
- **Data Preprocessing:** Handled missing values and normalized the data.
- **Clustering Analysis:** Applied KMeans clustering to uncover distinct customer segments.
- **Insights:** Provided actionable insights into customer behavior for better marketing strategies.

## Results
- Customers were grouped into three distinct clusters based on behavioral similarities.
- Insights from the clusters provided actionable recommendations for marketing strategies.

## Dataset
The dataset used for this project is available on Kaggle. Please download it from the following [link](https://www.kaggle.com/datasets/imakash3011/customer-personality-analysis).
After downloading, ensure the dataset is placed in the same folder as the project.
